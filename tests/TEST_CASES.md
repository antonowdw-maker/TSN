# Ручные тест-кейсы: Telegram-ассистент ТСН (демо)

Проверять в живом боте после деплоя. Базовый путь: `/start` → inline-меню.

## Команды и меню

**TC-01: Приветствие (`/start`)**  
Вход: `/start`  
Ожидание: приветствие с названием ТСН и главное inline-меню (4 раздела)

**TC-02: Справка (`/help`)**  
Вход: `/help`  
Ожидание: список команд, включая `/dispatcher` и `/schedule`

**TC-03: FAQ через команду (`/faq`)**  
Вход: `/faq`  
Ожидание: список категорий и кнопка **«Главное меню»**

## Поиск по тексту

**TC-04: Оплата взносов**  
Вход: «Как оплатить взносы?»  
Ожидание: ответ из FAQ; кнопки **«Популярные вопросы»** и **«Главное меню»**

**TC-05: Диспетчерская**  
Вход: «номер диспетчерской»  
Ожидание: ответ из FAQ или подсказка `/dispatcher`

**TC-06: Шумные соседи**  
Вход: «соседи шумят ночью»  
Ожидание: ответ с рекомендациями

**TC-07: Показания счётчиков**  
Вход: «показания счетчиков»  
Ожидание: ответ `meter_readings`

**TC-08: Нерелевантный вопрос без AI**  
Вход: «как приготовить борщ» (без `OPENAI_API_KEY`)  
Ожидание: «не нашёл ответ» + кнопки навигации

## Разделы главного меню

**TC-09: Контакты**  
Путь: `/start` → **Контакты** (или `/contacts`)  
Ожидание: контакты правления; кнопка **«Главное меню»**

**TC-10: Диспетчерская**  
Путь: `/start` → **Диспетчерская** (или `/dispatcher`)  
Ожидание: телефоны и режим; кнопка **«Главное меню»**

**TC-11: График работы**  
Путь: `/start` → **График работы** (или `/schedule`)  
Ожидание: расписание сотрудников; кнопка **«Главное меню»**

## Навигация FAQ (inline)

**TC-12: Список категорий**  
Путь: `/start` → **Популярные вопросы**  
Ожидание: все категории + **«Главное меню»**

**TC-13: Список вопросов категории «Коммунальные услуги»**  
Путь: **Популярные вопросы** → **Коммунальные услуги**  
Ожидание: список вопросов; кнопки **«К категориям»** и **«Главное меню»**

**TC-14: Список вопросов — все категории**  
Повторить TC-13 для каждой категории FAQ  
Ожидание: на экране вопросов категории всегда есть **«Главное меню»**

**TC-15: Ответ на вопрос из FAQ**  
Путь: **Популярные вопросы** → любая категория → любой вопрос  
Ожидание: текст ответа; кнопки **«К вопросам категории»** и **«Главное меню»**

**TC-16: Возврат в главное меню из FAQ**  
Путь: FAQ → категория → **Главное меню**  
Ожидание: приветствие `/start` и 4 раздела меню

**TC-17: Возврат к категориям**  
Путь: FAQ → категория → **К категориям**  
Ожидание: снова список категорий с **«Главное меню»**

**TC-18: Полный цикл FAQ**  
Путь: меню → FAQ → «Взносы и оплата» → вопрос → **К вопросам категории** → **К категориям** → **Главное меню**  
Ожидание: на каждом шаге навигация работает без тупиков

## AI (опционально)

**TC-19: Вопрос вне FAQ с AI**  
Условие: задан `OPENAI_API_KEY`  
Вход: сложный вопрос вне базы  
Ожидание: ответ AI + кнопки **«Популярные вопросы»** и **«Главное меню»**
